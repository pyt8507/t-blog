# t-blog
A blog based on JavaServer pages (JSP) technology

#### 项目背景
在今天，微博、微信等社交软件成为主流，个人博客不再那么流行，但仍有不少人喜欢经营自己的博客，在上面分享经验，记录生活。本次项目就将采用JSP + Servlet + Javabean + JDBC + MySQL方案开发一个轻量级的博客网站，实现博客网站的主要功能。

#### 开发环境
服务器：apache-tomcat-9.0.45

编辑器：IntelliJ IDEA 2020.3.3 x64

数据库：mysql-8.0.23-winx64

#### 实现功能
* 注册
* 登录
* 浏览文章
* 阅读和评论文章
* 个人资料
* 文章管理
* 发布文章
* 收藏文章
* 用户管理
